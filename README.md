# AccessNova – مساعد الوصول الذكي

منصة مدعومة بالذكاء الاصطناعي تساعد كبار السن وذوي الإعاقة على إنجاز المهام بسهولة عبر الصوت أو النص، باستخدام تقنيات Amazon Nova Sonic للتعرف على الكلام.

## ⭐ المميزات
- التعرف على الصوت باستخدام Nova Sonic  
- تحويل الكلام إلى نص بدقة عالية  
- تنفيذ أوامر المستخدم  
- تحويل النص إلى صوت  
- واجهة بسيطة وسهلة الاستخدام  
- دعم كامل للغة العربية  

## 🧠 كيف يعمل؟
يعتمد AccessNova على:
- MediaRecorder لتسجيل الصوت  
- Amazon Nova Sonic API لتحويل الصوت إلى نص  
- SpeechSynthesis لتحويل النص إلى صوت
